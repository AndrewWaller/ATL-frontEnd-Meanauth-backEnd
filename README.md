### Version
Beta         //1.0.0


## Usage


### Installation

#### Inside the root folder

Install the dependencies

```sh
$ npm install
```
Run server

```sh
$ npm start
```


#### Inside the angular-src folder

Install the dependencies

```sh
$ npm install
```
Run client-side

```sh
$ ng serve
```
